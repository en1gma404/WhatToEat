# Yemek Baku 🍽️

Не знаешь что поесть сегодня в Баку? Одна кнопка — и приложение выберет за тебя.

## О проекте

Yemek Baku — мобильное приложение для жителей Баку, которое случайно выбирает ресторан когда не можешь решить где поесть.

## Функционал

- Случайный выбор ресторана по кнопке
- Список ресторанов Баку с типом кухни
- Простой и быстрый интерфейс

## Стек

- React Native + Expo
- TypeScript

## Запуск

```bash
git clone https://github.com/твой-юзернейм/yemek-baku
cd yemek-baku
npm install
npx expo start
```
